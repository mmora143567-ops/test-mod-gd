# 6 Orbs & 3 Triggers Pack

Advanced custom gameplay mechanics pack for Geometry Dash.

## Features

* **Rewind Orb:** Features an adjustable 1-20 second slider configuration via Edit Object properties.
* **5 Custom Physics Orbs:** Mid-air interactive objects altering gravity, scale, and movement.
* **Gamemode Toggles:** Dynamic environment switch objects between Normal and Platformer modes.
* **Sudden Death Trigger:** Restrictive countdown timers designed specifically for platformer sections.
